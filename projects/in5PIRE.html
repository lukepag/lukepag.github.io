in5PIRE

Financial Forecasting Web App for Spire Hawaii LLP

View Source Live Demo

Overview

in5PIRE is a full-stack web app built for Spire Hawaii LLP to help organizations model, visualize, and manage financial forecasts. It provides data-driven dashboards, multiple profile management, and multi-year projection graphs (4-year, 8-year, 12-year) to support business decisions.

Tech Stack

  • Frontend: React, React Bootstrap, Uniforms
  • Backend: Meteor (Node.js full-stack framework)
  • Testing & CI: TestCafe, ESLint, GitHub Actions
  • Version Control: GitHub Projects, Issue-Driven Development
